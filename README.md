# Cinelog-java
Desafio técnico - cinelog
Funcionalidades
CRUD Completo: Cadastrar, Listar, Buscar por ID, Editar e Remover Filmes.
Favoritos: Adição de múltiplos filmes aos favoritos via IDs separados por vírgula.
Diferenciais Implementados:
    *   Busca de filmes por título (case-insensitive).
    *   Listagem ordenada por nota (decrescente).
    *   Filtro inteligente por gênero.

Como Compilar e Executar
Pré-requisitos: Certifique-se de ter o JDK (Java Development Kit) instalado na sua máquina (Versão 8 ou superior).
Clone ou baixe o repositório.
Abra o seu terminal/prompt de comando e navegue até a pasta raiz onde os arquivos .java estão salvos.
Execução deve ocorrer na classe Main.java
